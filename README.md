# Password Generator

A simple random password generator built using React and Tailwind CSS.

This project allows users to generate secure passwords with custom length and optional numbers and symbols.

---

## Live Demo - https://wasim2934.github.io/Password-Generator/


## Features

- Generate random passwords
- Adjust password length
- Include numbers
- Include symbols
- Copy password to clipboard
- Save generated passwords
- Reset settings

---

## Tech Stack

- React JS
- Tailwind CSS
- JavaScript
- Vite

---

## How It Works

- User selects password length
- User can choose whether to include:
  - Numbers
  - Symbols
- App generates a random password based on selected options
- Password can be copied or saved
